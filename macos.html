MacOS Guide

Getting Started with Your Mac

Basic Navigation

+

MacOS uses the Dock at the bottom of the screen for quick access to apps, and the menu bar at the top for system controls.

User Guide

+

macOS is the operating system used on Apple’s Mac computers, known for its sleek design and ease of use. To get started, familiarize yourself with the Dock at the bottom of the screen—it holds your most-used apps and the Finder, which helps you browse files and folders. The Menu Bar at the top changes based on the app you're using and always includes the Apple logo, where you can access system settings and shut down your Mac. Use Spotlight (press Command + Space) to quickly search for files, apps, or information. The System Settings (found via the Apple menu or Dock) lets you customize things like Wi-Fi, sound, display, and more. Finally, macOS comes with built-in apps like Safari (web browser), Mail, Notes, and Calendar to help you get things done right away.

Connecting to WiFi

+

Click the WiFi icon in the menu bar, select your network, and enter the password when prompted.

Installing Software

+

Use the App Store (found in the Dock) to install new applications. You can also download apps from the internet.

System Updates

+

Click the Apple menu > System Preferences > Software Update to check for and install system updates.
